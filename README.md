# 🏠 Smart House Price Predictor

Internship Project

## Overview
A machine learning web application that predicts residential property prices based on housing characteristics.

## Features
- House Price Prediction
- Random Forest Regression
- Interactive Dashboard
- Real-Time Results
- Model Performance Metrics

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

## Project Structure
```
House-Price-Prediction/
│
├── app.py
├── train_model.py
├── requirements.txt
├── README.md
├── data/
│   └── housing.csv
└── models/
    └── model.pkl
```

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python train_model.py
streamlit run app.py
```
